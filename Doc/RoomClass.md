# RoomClassについてのドキュメント

# 1. 目的
- pyroomacousticを用いた音響シミュレーションを行いやすくする

- シミュレーション条件の変更が容易である
# 2. 概要
- 音声信号のディレクトリと雑音信号のディレクトリ(または単一ファイル)を指定すると以下の音声がシミュレーションで生成される
1. 教師信号 (雑音なし，残響なし) : 全てのタスクにおける教師信号
2. 雑音信号 (雑音あり，残響なし) : 雑音抑圧用の教師信号
3. 残響信号 (雑音なし，残響あり) : 残響除去用の教師信号
4. 雑音+残響信号 (雑音あり，残響あり) : 雑音抑圧と残響除去を同時実行するための教師信号

- 実験の条件としては以下の項目をパラメータとしてConfig.jsonに設定する
1. 音源の位置
2. マイクアレイの形状 (マイク数, マイク間の距離, 線形 / 円形)
3. マイクアレイの中心位置 
4. 部屋の形状 
5. 残響時間 
6. 雑音の位置 
7. 雑音のSNR
